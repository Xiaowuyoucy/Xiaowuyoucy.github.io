# java继承格式

##### 子类定义语法:

```java
public class 子类名 extends 父类名称{
	//代码块
}
```





```java
package xiaochenyan.top.fuZiLei;

public class demo01{
    public static void main(String[] args) {
        Teacher xiaoMing = new Teacher();

        xiaoMing.setAge(23);
        xiaoMing.setName("XiaoMing");
        xiaoMing.show();
    }
}
```



子类会继承父类的公有成员函数或公有变量.

```java
package xiaochenyan.top.fuZiLei;

public class Teacher extends YuanGong{
    Teacher(){
        show();
    }
}

```



##### 父类

```java
package xiaochenyan.top.fuZiLei;

public class YuanGong{
    public String name;
    public int age;

    public YuanGong(String name, int age) {
        this.name = name;
        this.age = age;
    }

    public YuanGong() {
        name = null;
        age = 0;
    }

    public String getName() {
        return name;
    }

    public void setName(String name) {
        this.name = name;
    }

    public int getAge() {
        return age;
    }

    public void setAge(int age) {
        this.age = age;
    }
    //输出公有成员变量
    public void show()
    {
        System.out.println("姓名:"+name);
        System.out.println("年龄:"+age);
    }
}

```
