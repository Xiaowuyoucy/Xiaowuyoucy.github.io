# 其他进制的数字

在JS中,如果需要表示16进制的数字,则需要以0x开头.

```js
var a = 0x12334;
console.log(a);
```

如果需要表示8进制,则用0开头

```js
var a = 010;
console.log(a);
```

如果需要表示二进制,用0b开头.但不是所有浏览器都支持.

```js
var a = 0b010;
console.log(a);
```

有些浏览器会把"070"这种字符串当成10进制

```js
var a = "070";
parseInt(a);//有些浏览器会把这种字符串当成10进制
//解决方法
parseInt(a,8);
```
