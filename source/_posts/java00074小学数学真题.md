# java小学数学真题

计算从-10.8到5.9之间,绝对值大于6或者小于2.1的整数有多少个?

```java
package xiaochenyan.top.Demo;

public class Main21{
    public static void main(String[] args) {
        int count = 0;
        double min = -10.8;
        double max = 5.9;

        for (double i = min; i < max; i++) {
            double num = Math.ceil(i);
            double abS = Math.abs(num);
            if (abS > 6 || abS < 2.1)
            {
                ++count;
            }
        }
        System.out.println(count);
    }
}

```
