## nslookup把域名转换为ip

`nslookup 域名`

```
nslookup www.baidu.com
```

```
root@yc:/home/yxc19980620c/my# nslookup www.baudu.com
Server:		127.0.0.53
Address:	127.0.0.53#53

Non-authoritative answer:
Name:	www.baudu.com
Address: 47.91.20.194


```
