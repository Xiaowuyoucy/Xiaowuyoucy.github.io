# 第一个c++程序 hello world

```c++
#include<iostream>

using namespace std;

int main(int argc, char* argv[])
{

	cout << "hello world" << endl;
	
    
    int a;
    cin >> a;
    
	return 0;
}
```

cout 表示输出流，将hello world流到终端

endl 表示换行

cin 表示输入流，从键盘输入内容流到变量a中
