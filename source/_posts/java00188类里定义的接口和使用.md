# 类里定义的接口和使用

### 格式:

```java
public class 类名{
	//在类里面定义接口
    修饰符 interface 接口名{
		//抽象方法;
    }
}
```

```java
public class TestA{
	//在类里面定义接口
    private interface A{
        public abstract void show();
    }
}
```

### 实现接口:

##### 在类方法中用new实现,通过多态接收

```java
public class TestA{
	//在类里面定义接口
    
    	//如果接口修饰符石private,那么此接口只能在本类中使用.
    	public interface A{
        	public abstract void show();
    	}
        public void show(){
        A a = new A(){

            @Override
            public void show() {
                System.out.println("111111");
            }
        };
        a.show();
    }
}
```

##### 在new的时候实现

```java
package com.itheima.demo11;

public class DemoClass{
    public static void main(String[] args) {
        TestA.A a = new TestA.A(){
            @Override
            public  void show(){
                System.out.println("我是TestA类中的A接口");
            }
        };

       a.show();
    }
}

```

##### 在其他类中实现

```java
package com.itheima.demo11;

public class TestB implements TestA.A{
  @Override
    public void show() {

    }
}

```

```java
public class TestA{
	//在类里面定义接口
    private interface A{
        public abstract void show();
    }

   	 public void show() {

        }
    }
}
```
