# JS_Boolean类型

### Boolean布尔值

#### 布尔值有两个,主要用来做逻辑判断

#### `true`

#### &emsp;&emsp;表示真

#### `false`

#### &emsp;&emsp;表示假

##### 使用`typeof`检查一个布尔值时,会返回一个`boolean`

```html
<!DOCTYPE html>
<html>
	<head>
		<meta charset="UTF-8">
		<title></title>
		<script>
			var isBingo = false;
			
			console.log(typeof false);
			console.log(typeof true);
			console.log(isBingo);  
		</script>
	</head>
	<body>
	</body>
</html>

```
